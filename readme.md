# Frontend for the [Book Store API](https://github.com/opermaster/BookStoreRest)

Based on HTML, CSS, and JavaScript.

## Overview

This project provides a web interface for buying and selling books. Users can browse the catalog, purchase books, or sell their own by adding new items and authors.

---

## Books Page

The Books page displays all books available in the store. Users can:

* Browse books
* Buy books
* Search for books using the text input field

## Personal Page

The Personal page is used to manage orders for sellers and purchases for buyers. It also allows adding new books to the store. When adding a book, users can either select an existing author or create a new one.
